# CLIPR — Local Video Editor

**CLIPR** adalah aplikasi pemotong video berbasis *client-side* yang berjalan sepenuhnya di dalam browser. Dirancang untuk kecepatan, privasi, dan kemudahan penggunaan tanpa perlu proses *upload* ke server.

## 🚀 Fitur Utama
* **100% Private & Secure**: Tidak ada video yang diunggah ke server. Semua proses dilakukan secara lokal di perangkat Anda.
* **Smart Cropping**: Mendukung rasio aspek media sosial (9:16, 1:1, 4:5, 16:9) dengan antarmuka *drag-and-drop*.
* **Speed Control**: Atur kecepatan video (0.5x, 1x, 1.5x, 2x) untuk efek dramatis atau *fast-forward*.
* **Queue System**: Ekspor hingga 5 potongan video sekaligus dalam satu alur kerja.
* **No Install Required**: Jalankan langsung di browser modern (Chrome, Edge, Firefox).

## 🛠 Teknologi
* **Frontend**: HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (Vanilla).
* **Core API**: `MediaRecorder` API, `Canvas` API, `Web Audio` API.
* **Deployment**: Dihosting di https://clipe.netlify.app/
